Project 2 
Gossip and Pushsum Simulator

In order to build the project and run the simulation run the following command:

    sbt "run numNodes topology algorithm"

numNodes is a positive integer.

topology is one of the following:
    full
    line
    3D
    imp3D

algorithm is one of the following:
    gossip
    push-sum

Please include the quotation marks as well.

All features are implemented and working. Output will be displayed in the terminal.
The largest network we tested for all cases was 27000 nodes.
Report.pdf is included in this directory.
